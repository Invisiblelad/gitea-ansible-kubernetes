About Gitea

Gitea is a lightweight, self-hosted Git service, providing a powerful and streamlined code hosting solution. It offers a simple UI, support for issue tracking, code review, and CI/CD integration, making it a popular choice for DevOps workflows.

Key Features

  • Lightweight and self-hosted
  • Easy-to-use interface
  • Integrated issue tracking and code review
  • CI/CD compatible with popular DevOps tools
  • Supports SSH, HTTP/HTTPS protocols
